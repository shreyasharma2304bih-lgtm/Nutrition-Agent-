# 🥗 Nutrition Agent using IBM Skills

An AI-powered **Nutrition Agent** developed using **IBM Skills** that assists users in making healthier food choices by providing nutrition information, meal recommendations, calorie guidance, and healthy lifestyle tips through an interactive conversational interface.

---

## 📌 Project Overview

The Nutrition Agent is an intelligent virtual assistant designed to answer nutrition-related questions and promote healthy eating habits. It uses IBM Skills and Natural Language Processing (NLP) to understand user queries and provide relevant nutritional guidance.

The agent can help users:
- Learn about the nutritional value of foods
- Receive healthy meal suggestions
- Estimate daily calorie requirements
- Get hydration recommendations
- Understand BMI and healthy lifestyle practices
- Receive general dietary advice

## 🚀 Getting Started

### Prerequisites

Before using the project, ensure you have:

- IBM Cloud Account
- IBM Skills access
- Internet connection
- Modern web browser

---

## ⚙️ Setup Instructions

1. Sign in to IBM Cloud.
2. Open IBM Skills.
3. Create a new Nutrition Agent skill.
4. Configure intents and entities.
5. Design the dialog flow.
6. Deploy the skill.
7. Test the chatbot with sample queries.
